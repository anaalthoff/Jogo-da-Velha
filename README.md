# Jogo da Velha (Tic Tac Toe)

Um projeto simples em HTML, CSS e JavaScript desenvolvido há alguns anos, que implementa o clássico Jogo da Velha para dois jogadores, com interface gráfica e exibição de vencedores.

## Funcionalidades

- Entrada de nomes para Jogador X e Jogador O
- Alternância automática entre turnos (X e O)
- Destaque visual da linha vencedora
- Indicação de vencedor com mensagem personalizada (com nome do jogador)
- Detecção de empate (quando todos os espaços são preenchidos)
- Botão de Reset para reiniciar a partida

🎵 Música de fundo (Clair de Lune, Debussy) para tornar a experiência mais agradável

## Interface

- O tabuleiro é construído com grid layout (3x3).
- Cada célula é representada por um botão clicável.
- O resultado da partida é exibido logo abaixo do tabuleiro.

## Como Executar

1. Baixe ou clone este repositório.
2. Certifique-se de ter os arquivos image.jpg e o áudio .mp3 no mesmo diretório do jogoDaVelha.html.
3. Abra o arquivo jogoDaVelha.html em qualquer navegador moderno.
4. Insira os nomes dos jogadores e clique em Iniciar Partida.

## Exemplo de Uso

1. Informe os nomes dos jogadores.
2. Clique em uma célula do tabuleiro para realizar uma jogada.
3. O sistema alternará automaticamente entre X e O.
4. Assim que houver um vencedor ou empate, a mensagem será exibida.
5. Clique em Reset para começar novamente.

## Tecnologias Utilizadas

- HTML5
- CSS3 (Grid Layout, estilização de botões e interface)
- JavaScript (lógica do jogo, verificação de vencedor/empate, manipulação do DOM)
